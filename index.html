What is this?

This ratio calculator helps you solve an incomplete ratio equasion where one or two values are missing. Use it to scale a ratio up or down, find a ratio's lowest terms or test if two ratios are equivalent.

Usage

Plug in your known values and hit the "Solve for missing value(s)" button to calculate the missing values. You must begin with at least both values in the left fraction filled out.

Starting with 2 values

Starting with two values will automatically populate the right fraction with an equivalent fraction, as expressed in lowest terms. The lowest terms are always displayed below the main controls as well.

Starting with 3 values

Starting with three values lets you find the remaining missing value to make your ratios equivalent.

Starting with 4 values

Starting with all four values included is a handy way to test a complete ratio for equivalency. Equivalency is always shown below the main controls regardless of how many values you begin with.
